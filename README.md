### GERENCIADOR DE TAREFAS
---
### SOBRE O PROJETO

Desenvolvimento de uma API RESTful para gerenciamento de tarefas, utilizando ASP.NET Core. Essa aplicação permite criação, leitura, atualização e exclusão de tarefas.

---
### TECNOLOGIAS USADAS
- C#
- ASP.NET
- SQLITE
- ENTITY FRAMEWORK
